# ☕ FoodiePlace Café Website

A modern, responsive, and ultra-premium café website built using **React.js** and custom CSS.  
Designed for a real-world restaurant/café experience with smooth UI, elegant sections, and mobile-first responsiveness.

## ✨ Features

- ⚡ Fully responsive modern UI
- 🎨 Ultra-premium café-style design
- 🍽️ Hero section with strong branding
- 📍 Location with embedded Google Map
- 🕒 Opening hours section
- ⭐ Customer testimonials
- 📸 Image gallery with hover effects
- 🎁 Special offers section
- 📞 Contact / reservation form
- 🧑‍🍳 About & Why Choose Us sections
- 📱 Mobile-friendly hamburger menu
- 🎯 Smooth section navigation

---

## 🧑‍💻 Tech Stack

- React.js (Functional Components)
- JavaScript (ES6+)
- CSS3 (Custom styling, Flexbox, Grid)
- Google Maps Embed API
